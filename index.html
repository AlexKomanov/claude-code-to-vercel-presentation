Slide 01 — Introduction

What is Claude Code?

🤖
Claude Code
An AI-powered command-line tool by Anthropic that helps you build software by talking to it — like texting a brilliant developer friend.
🧩
Skills & Agents
Special plug-ins you can add to Claude Code. Skills teach it new tricks. Agents let it tackle complex multi-step tasks automatically.
🎨
Presentation Builder
A special skill that turns your ideas into stunning, ready-to-share HTML presentations — no design experience needed.

WHY USE AI FOR PRESENTATIONS?

Instead of spending hours in PowerPoint or Keynote, you simply describe what you want and Claude Code builds a beautifully designed, interactive HTML presentation for you — in seconds. Then you share it as a live website anyone can visit from anywhere on Earth.

Step 01
1

Install Claude Code

Claude Code is a command-line tool — a program you run by typing commands. Setting it up takes just a couple of minutes!

  • 📦 You need Node.js installed first — it's free at nodejs.org
  • 🔑 Get your free API key at console.anthropic.com
  • 💻 Open your Terminal (Mac) or Command Prompt (Windows)
TERMINAL COMMANDS
$
npm install -g @anthropic-ai/claude-code
# Install Claude Code globally
$
claude
# Launch Claude Code
$
claude --version
# Verify it's working
⚠️

First time using a terminal? Don't worry! Just copy-paste these commands one at a time and press Enter.

Step 02
2

Ask Claude to Build It

With Claude Code running, you just type what you want in plain English. It's that simple — no coding required from your end!

  • 🎯 The presentation-builder agent handles all design decisions
  • ⚡ It generates a complete HTML file with animations and navigation in seconds
  • 📁 Your file is saved automatically to your Documents folder
Example Prompt — Type This in Claude Code
Using a presentation-builder agent, create a presentation about the history of space exploration
WHAT HAPPENS NEXT
1️⃣ Claude thinks through your topic

Step 04
4

Create a GitHub Repo

GitHub is like a cloud storage for code. Creating an account and repository (a project folder) is free and only takes 2 minutes.

  • 🌐 Go to github.com and sign up for a free account
  • ➕ Click the "+" icon → "New repository"
  • ✏️ Name it something like my-presentation
  • 🔓 Set it to Public so Vercel can access it
  • ✅ Click "Create repository" — done!

Step 05
5

Upload to GitHub

Now we use git — a tool for saving and uploading code — to send your HTML file to GitHub. Run these commands in your terminal, one by one.

📁 First, navigate your terminal to the folder with your HTML file — usually your Documents folder.

GIT COMMANDS — RUN IN ORDER
$
git init
# Initialize git in this folder
$
git add .
# Stage all files for upload
$
git commit -m "Add presentation"
# Save a snapshot
$
git remote add origin <your-repo-url>
# Link to GitHub
$
git push -u origin main
# Upload to GitHub!
💡 Your repo URL looks like: https://github.com/you/my-presentation.git
